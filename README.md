# sdd-hackaton-25-heart-risk
Our team's repository for the sdd's mini-hackathon

## Requirements
⚠️ This repo uses Git LFS to store large model files.  
To clone properly, make sure you have Git LFS installed and running.
### 1 - Install Git LFS
```bash
sudo apt install git-lfs
```
### 2 - - Initialize Git LFS (once per machine):
```bash
git lfs install
```
